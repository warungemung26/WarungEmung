🔹 Cara Belanja Satu Jenis Barang

Di setiap Barang ada tombol Qty - / + untuk menambah jumlah produk, setelah itu pilih tombol "Pesan Sekarang". Cocok jika Anda hanya ingin membeli 1 jenis barang. Setelah klik, WhatsApp terbuka otomatis dengan isi pesan siap kirim.

🔹 Cara Belanja Banyak Jenis Barang

Gunakan tombol Qty - / + untuk menambah jumlah produk, lalu klik + ke Keranjang 🛒. Setelah selesai memilih, klik tombol 🛒 lalu klik Pesan untuk kirim lewat WhatsApp.

🔹 Kalau Produk Tidak Ada di Website

Gunakan tombol Request Stok di halaman utama. Anda bisa ketik manual nama produk yang dicari. Admin IT akan mempertimbangkan & menambahkan produk tersebut secepat mungkin.

🔹 Cara Mencari Produk

Ketik nama produk di kolom Cari Produk / Search di bagian atas halaman. Sistem akan menampilkan hasil secara instan.

🔹 Cara Pilih Kategori

Gunakan menu Kategori untuk menampilkan hanya produk sesuai kelompok (contoh: Sembako, Minuman, Mie Instan).

🔹 Bantuan & Layanan WhatsApp

WhatsApp hanya untuk cek ketersediaan produk & transaksi pemesanan.
Pertanyaan di luar transaksi silakan gunakan tombol email di atas.

FAQ Lengkap

1. Cara Pesan Satu Produk

• Pilih jumlah dengan tombol Qty - / +
• Klik tombol Pesan Sekarang → langsung ke WhatsApp admin

2. Cara Pesan Banyak Produk

• Tentukan Qty lalu klik Tambahkan ke Keranjang
• Ulangi untuk produk lain
• Klik Lihat Keranjang untuk checkout
• Klik Kosongkan Keranjang untuk reset

3. Jika Produk Tidak Tersedia

Klik Request Stok dan ketik manual nama produk yang dicari.

4. Masukan & Pertanyaan Lain

Gunakan tombol Kirim Email untuk pertanyaan non-transaksi → warung.emung26@gmail.com
